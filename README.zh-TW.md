# 梅花易數 Meihua Yishu

[![License: CC BY-NC-SA 4.0](https://img.shields.io/badge/License-CC%20BY--NC--SA%204.0-lightgrey.svg)](https://creativecommons.org/licenses/by-nc-sa/4.0/)

**[English](README.md)** | **[繁體中文](README.zh-TW.md)**

**梅花易數**（Plum Blossom Numerology）是相傳為宋代邵雍（邵康節）所創的中國傳統易學占卜術。此專案提供一個專業的梅花易數占卜系統，可作為 Claude AI 的 Skill 使用。

## 功能特色

### 起卦方式
- **時間起卦** — 以當前或指定時間起卦
- **數字起卦** — 用數字來起卦
- **聲音起卦** — 以聽到的聲音次數起卦
- **顏色起卦** — 根據所見顏色對應五行起卦
- **尺寸起卦** — 以物品尺寸數字起卦
- **方位起卦** — 根據來人或事物方位起卦

### 解卦功能
- **體用生克分析** — 解讀卦象中的體卦與用卦關係
- **通關化解** — 分析五行通關緩解克象的技巧
- **六十四卦解讀** — 詳細解釋卦辭與爻辭
- **動爻變卦推演** — 分析變卦的吉凶
- **卦氣旺衰** — 根據季節判斷體卦強弱
- **應期推斷** — 預測事情應驗的時間

### 專門占法
- **十八類分占** — 婚姻、疾病、求財、出行等專門占法
- **十應判斷** — 天時、地理、人事等外應分析
- **八卦萬物類象** — 提供現代與傳統的類象對照
- **測字占卜** — 拆字、相字、字占

### 智能輔助
- **照片分析** — 上傳照片，AI 分析環境外應
- **環境感知** — 描述周遭聲音、顏色、人物，增強斷卦

## 使用方式

### 作為 Claude Skill 使用

將此資料夾放入你的 Claude Skills 目錄即可使用。

### 使用 Python 計算工具

```bash
# 以當前時間起卦
python scripts/meihua_calc.py time

# 以數字起卦（兩個數字）
python scripts/meihua_calc.py num 6 8

# 以數字起卦（三個數字，第三個為動爻）
python scripts/meihua_calc.py num 6 8 3
```

## 核心原理

梅花易數以**先天八卦數**起卦：

| 卦 | 數 | 五行 | 符號 |
|---|---|------|------|
| 乾 | 1 | 金 | ☰ |
| 兌 | 2 | 金 | ☱ |
| 離 | 3 | 火 | ☲ |
| 震 | 4 | 木 | ☳ |
| 巽 | 5 | 木 | ☴ |
| 坎 | 6 | 水 | ☵ |
| 艮 | 7 | 土 | ☶ |
| 坤 | 8 | 土 | ☷ |

### 體用論

- **體卦**：主體、自己、問卦者
- **用卦**：客體、事情、外在環境
- **互卦**：事情發展過程
- **變卦**：最終結果

### 吉凶判定

| 情況 | 吉凶 | 說明 |
|------|------|------|
| 用生體 | 大吉 | 有進益之喜，外力相助 |
| 體克用 | 吉 | 我制約對方，可成事 |
| 體用比和 | 吉 | 五行相同，和諧順利 |
| 用克體 | 凶 | 受制於人，不利 |
| 體生用 | 耗洩 | 付出多回報少 |

### 通關化解

當體用相克時，若互卦或變卦中有「通關」元素，可緩解凶象：

| 克關係 | 通關元素 |
|--------|----------|
| 金克木 | 水 |
| 木克土 | 火 |
| 土克水 | 金 |
| 水克火 | 木 |
| 火克金 | 土 |

## 占卜原則

1. **無疑不卜** — 沒有具體問題不占
2. **筮不過三** — 同一問題不重複占卜超過三次
3. **不動不占** — 無事不占
4. **依理推斷** — 卦象需結合實際情況解讀

# 核心
易=變，能變則通，困守則凶。占卜目的是指引如何順勢而變，而非預定吉凶。
# 類型
吸引子:自己好變出去差→留, 排斥子:自己差變出去好→走, 陷阱:自己差鄰居也差→慎, 福地:文本偏吉→守, 困境:文本偏凶→變, 一般:中等→觀
# 各策略【下一步】行動文字（meihua_calc.py 自動輸出，須完整呈現）
留|維持現狀，不宜改變。目前位置有利，變動反而損失。
走|積極改變，離開當前狀態。此位置不利久留，宜主動求變。
守|穩守不動，靜觀其變。位置尚可，不主動出擊，等待時機。
變|必須改變，不變則困。當前困境需主動突破，猶豫更糟。
慎|謹慎行事，小心陷阱。周圍環境不佳，任何動作都要三思。
觀|觀察局勢，再做決定。情況中等，需要更多信息才能判斷。
